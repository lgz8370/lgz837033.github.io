# 更新git密码

使用sourcetree时登录git账号,git账号修改了密码后sourcetree在提交/拉取代码时只会报错,并不会弹出git登录窗口让用户重新登录.

# 解决方法: windows

进入以下目录,删除passwd文件

C:\Users\USERNAME\AppData\Local\Atlassian\SourceTree
1

删除文件后,再次拉取/提交代码就会弹出git登录窗口
