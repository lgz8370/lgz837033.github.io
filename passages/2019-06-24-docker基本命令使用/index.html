记录docker常用命令

# 1. 查看本地镜像列表
docker images
1
# 2. 查看运行中的容器
docker ps
1
# 3. 查看所有容器(包括未运行的)
docker ps -a
1
# 4. 删除镜像
docker rmi <镜像ID>
1
# 5. run命令
  • docker run [参数] [镜像id] | [镜像名]:[镜像标签] [命令]

-i 表示容器的输入终端保持打开,不直接关闭 -t 表示开启一个虚拟终端, 绑定到标准输入上 -d 表示让该容器在后台运行 --name 给该容器起一个容器名，如果没有此参数，则有一个默认的容器名 --rm 停止该容器时，把容器删除掉

# 6. 使用镜像运行容器
docker run -it --name <容器名称> -d -p <port:port> <镜像ID>
1
# 7. 启动容器
docker start <容器ID>
1
# 8. 停止容器
docker stop <容器ID>
1
# 9. 删除容器
docker rm <容器ID>
1
# 10. 查看docker的所有容器运行状态
docker stats
1
# 11. 进入容器里面
docker exec -it <容器ID> /bin/bash
1
# 12. 查看容器运行日志(类似 tail -f 命令)
docker logs -f <容器ID/名称>
1
# 13. 构建镜像
  • 进入存放dockerfile目录,执行构建命令
docker build -t [镜像名]:[镜像标签] [存放dockerfile的目录]
1
